## Available Scripts

1. **install_anydesk.sh**
   - Installs AnyDesk, a remote desktop application

2. **install_brave.sh**
   - Installs Brave Browser, a privacy-focused web browser

3. **install_chrome.sh**
   - Installs Google Chrome web browser

4. **install_etcher.sh**
   - Installs Etcher, a tool for flashing OS images to SD cards & USB drives

5. **install_github_desktop.sh**
   - Installs GitHub Desktop, a GUI for Git and GitHub

6. **install_keepassxc.sh**
   - Installs KeePassXC, an open-source password manager

7. **install_obs_studio.sh**
   - Installs OBS Studio, a free and open source software for video recording and live streaming

8. **install_teamviewer.sh**
   - Installs TeamViewer, a remote access and support software

9. **main.sh**
   - Provides a menu-driven interface to install any or all of the above tools

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/Shwet-Kamal-Singh/linux-Installer-Hub.git
   cd linux-Installer-Hub/utilities
   ```

2. Make scripts executable:
   ```bash
   chmod +x *.sh
   ```

3. Run the main script:
   ```bash
   ./main.sh
   ```

4. Follow on-screen prompts to install desired tools.
